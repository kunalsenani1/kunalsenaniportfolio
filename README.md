# Kunal Senani Portfolio

Premium cinematic single-page portfolio website for Kunal Senani, built with Next.js, Tailwind CSS, Framer Motion, and lucide-react.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Replace placeholders

Project cards and the showreel use cinematic placeholders. Search for `Replace` comments in `components/PortfolioPage.tsx` to connect real thumbnails or video URLs.
